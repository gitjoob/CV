# Phishing Resistant MFA User Export

This README provides information about exporting users from the Phishing Resistant MFA Workbook in Azure.

## Overview

This documentation relates to the **Phishing-Resistant Passwordless Deployment (Preview)** workbook available in the Azure Portal. The workbook can be accessed at:
[Azure Portal - Phishing Resistant MFA Workbook](https://portal.azure.com/#blade/AppInsightsExtension/UsageNotebookBlade/ComponentId/Azure%20Active%20Directory/ConfigurationId/community-Workbooks%2FAzure%20Active%20Directory%2FPasswordless/WorkbookTemplateName/Phishing-Resistant%20Passwordless%20Deployment%20(Preview))

## Export Files

The export process generates **three separate CSV files**, one for each platform:

- **Windows.csv** - Contains user data for Windows devices
- **MacOS.csv** - Contains user data for MacOS devices
- **iOS.csv** - Contains user data for iOS devices

## Purpose

These exports help identify and track users who are using or eligible for phishing-resistant multi-factor authentication methods across different device platforms.
